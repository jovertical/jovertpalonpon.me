## Quick Start

1. Run `yarn install`.
2. Copy the environment file: `cp .env.example .env.build`.
3. Run `yarn dev` to start the development server.

## Testing

Run unit tests with:

```
yarn test:unit
```

## Linting

Run the linter with:

```
yarn test:lint
```
